# Proyecto Challenge 1 - Data Science Latam

Este proyecto fue desarrollado como parte del desafío de Data Science en el curso de Alura Latam.

## Descripción

El proyecto consiste en un análisis detallado de datos de 4 tiendas, para decidir cual de ellas debe vender e invertir en otro negocio.

## Tecnologías Utilizadas

- Python
- Pandas
- Matplotlib
- Git
- Colab

## Análisis Realizado

1. **Facturación Total de cada Tienda**: Se analizó la facturación de las tiendas para identificar cuál tiene mayores ingresos.
2. **Categorías de Productos Más Vendidas**: Identificación de las categorías más populares y menos populares por tienda.
3. **Promedio de Evaluaciones de Clientes**: Cálculo del promedio de las calificaciones de los clientes por tienda.
4. **Productos Más y Menos Vendidos**: Identificación de los productos más vendidos y menos vendidos por tienda.
5. **Costo de Envío Promedio**: Cálculo del costo promedio de envío de cada tienda.

## Instalación

Para ejecutar este proyecto en tu máquina local, sigue los siguientes pasos:

1. Clona este repositorio:

```bash
git clone https://github.com/pame-olguin/challenge1-data-science-latam.git

2. Acceda al directorio del proyecto:

cd challenge1-data-science-latam

3. Asegúrese de tener instaladas las dependencias necesarias. Si no las tienes, puedes instalar las bibliotecas que se usan en este proyecto utilizando:

pip install -r requirements.txt

Nota : Si no tienes el archivo requirements.txt, puedes instalar las librerías manualmente, que incluyen pandas, matplotliby cualquier otra dependencia que hayas utilizado.

4. Abra y ejecute el proyecto en su entorno preferido, como Jupyter, Colab o cualquier entorno que soporte Python.

Conclusión
Este proyecto proporciona un análisis detallado para ayudar a tomar decisiones sobre cuál tienda vender, con base en un análisis integral de su desempeño en ventas y operaciones.
